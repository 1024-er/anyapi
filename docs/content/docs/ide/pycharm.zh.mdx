---

## Step 1: 安装 AI 插件

1. 打开 PyCharm → 设置 → 插件
2. 搜索兼容 OpenAI 的 AI 插件
3. 安装并重启 PyCharm

## Step 2: 配置 API

1. 进入插件设置
2. 设置 **API Base URL**: `https://api.example.com/v1`
3. 输入你的 **API Key**
4. 选择你偏好的模型

## Step 3: 开始使用

使用 PyCharm 中的 AI 聊天面板获取编程辅助。
