## Step 1: Install Windsurf

1. Visit the Windsurf website to download the installer
2. Install and open Windsurf

## Step 2: Configure API

1. Open Settings
2. Navigate to AI configuration
3. Set **API Base URL**: `https://api.example.com`
4. Enter your **API Key**

## Step 3: Start Using

Windsurf provides AI-powered code editing, chat, and multi-file editing capabilities.
