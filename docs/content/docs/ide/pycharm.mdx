## Step 1: Install AI Plugin

1. Open PyCharm → Settings → Plugins
2. Search for an OpenAI-compatible AI plugin
3. Install and restart PyCharm

## Step 2: Configure API

1. Go to plugin settings
2. Set **API Base URL**: `https://api.example.com/v1`
3. Enter your **API Key**
4. Select your preferred model

## Step 3: Start Using

Use the AI chat panel in PyCharm to get coding assistance.
