## 什么是 Any API？

Any API 是专为开发者打造的 AI API 中转服务平台。提供稳定的网络线路、便捷的支付方式，让你轻松使用全球顶尖的 AI 编程助手。

### 支持的 AI 工具

- [Claude Code](/docs/installation/macos) — Anthropic 官方 AI 编程助手，支持终端交互、代码生成、项目理解（推荐）
- [Codex CLI](/docs/installation/macos) — OpenAI 推出的命令行 AI 编程工具，支持代码补全和生成
- [Gemini CLI](/docs/installation/macos) — Google 的 AI 编程助手，支持多模态理解和代码生成

### 快速上手四步走

1. **获取 API 密钥** — 注册并登录，在控制台获取你的专属密钥 → [前往获取](/console/token)
2. **安装 AI 工具** — 根据你的操作系统，安装对应的 AI 编程工具
3. **配置中转服务** — 设置环境变量，将 API 请求指向中转服务
4. **开始使用** — 启动工具，享受流畅的 AI 辅助编程体验
