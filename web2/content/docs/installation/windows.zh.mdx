## Step 1: 安装 Node.js 环境

**方法一：官网下载（推荐）**

1. 打开浏览器访问 [https://nodejs.org](https://nodejs.org)
2. 点击「LTS」版本进行下载
3. 下载完成后双击 .msi 文件
4. 按照安装向导完成安装，保持默认设置即可

**方法二：使用包管理器**

```bash
# Chocolatey
choco install nodejs

# Scoop
scoop install nodejs
```

<Callout>
建议使用 PowerShell 而不是 CMD。如果遇到权限问题，尝试以管理员身份运行。
</Callout>

**验证安装是否成功**

```bash
node --version
npm --version
```

## Step 2: 安装 Claude Code

```bash
npm install -g @anthropic-ai/claude-code
```

**验证 Claude Code 安装**

```bash
claude --version
```

## Step 3: 设置环境变量

**方法一：PowerShell 临时设置（当前会话）**

```powershell
$env:ANTHROPIC_BASE_URL = "https://api.example.com"
$env:ANTHROPIC_AUTH_TOKEN = "your_api_key"
```

**方法二：PowerShell 永久设置（用户级）**

```powershell
[System.Environment]::SetEnvironmentVariable("ANTHROPIC_BASE_URL", "https://api.example.com", [System.EnvironmentVariableTarget]::User)
[System.Environment]::SetEnvironmentVariable("ANTHROPIC_AUTH_TOKEN", "your_api_key", [System.EnvironmentVariableTarget]::User)
```

设置后需要重新打开 PowerShell 窗口才能生效。

**验证环境变量设置**

```powershell
echo $env:ANTHROPIC_BASE_URL
echo $env:ANTHROPIC_AUTH_TOKEN
```

## Step 4: 开始使用

```bash
claude
```

**在特定项目中使用**

```bash
cd C:\path\to\your\project
claude
```
