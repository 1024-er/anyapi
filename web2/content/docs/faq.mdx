## Q: Which models are supported?

We support Claude, OpenAI, Gemini and other mainstream models, synced with official releases.

## Q: How do I get an API key?

After logging in, go to Console → API Keys, and click "Create Key" to generate one. Keep your key secure.

## Q: What calling methods are supported?

The platform is compatible with the OpenAI API format. Simply replace the request URL with our proxy address to seamlessly integrate with ChatGPT Next Web, Lobe Chat, and other third-party clients.

## Q: What if I run out of credits?

Go to Console → Top Up to recharge, or contact the administrator for more credits.

## Q: How do I view usage history?

Go to Console → Logs to view model usage, token consumption, and cost details for each API call.

## Q: Is my code secure?

Code is transmitted via encryption. The platform does not store code content. Chat history is saved locally. Sensitive operations require confirmation.

## Q: How to troubleshoot connection issues?

1. Check that environment variables are correctly set
2. Confirm your API key is valid and has available credits
3. Verify the request URL is correct (don't forget the /v1 path)
