title: 常见问题
description: 关于 AI API 平台和工具的常见问题解答
---

## Q: 支持哪些模型？

支持 Claude、OpenAI、Gemini 等主流模型，与官方同步更新。

## Q: 如何获取 API 密钥？

登录后进入「控制台」→「API 密钥」页面，点击「创建密钥」即可生成。请妥善保管密钥，不要泄露给他人。

## Q: 支持哪些调用方式？

平台兼容 OpenAI API 格式，只需将请求地址替换为本站地址，即可无缝对接 ChatGPT Next Web、Lobe Chat 等第三方客户端和开发框架。

## Q: 额度用完了怎么办？

可前往「控制台」→「充值」页面进行额度充值，或联系管理员获取更多额度。

## Q: 如何查看使用记录？

进入「控制台」→「日志」页面，可查看每次 API 调用的模型、Token 用量和消费明细。

## Q: 代码安全吗？

代码通过加密传输，平台不存储代码内容。对话历史保存在本地。敏感操作需要确认。

## Q: 连接问题如何排查？

1. 检查环境变量是否正确设置
2. 确认 API 密钥有效且有可用额度
3. 检查请求地址是否正确（不要遗漏 /v1 路径）
