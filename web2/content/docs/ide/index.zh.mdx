## Claude Code CLI

**安装扩展**

1. 打开 VSCode，按 `Ctrl+Shift+X` / `Cmd+Shift+X`
2. 搜索「Claude Code for VS Code」
3. 安装 Anthropic 官方发布的扩展

**配置中转服务**

编辑 `~/.claude/settings.json`（Mac）或 `C:\Users\你的用户名\.claude\config.json`（Windows）：

```json
{
  "env": {
    "ANTHROPIC_AUTH_TOKEN": "替换为您的API Key",
    "ANTHROPIC_BASE_URL": "https://api.example.com",
    "CLAUDE_CODE_DISABLE_NONESSENTIAL_TRAFFIC": 1
  },
  "permissions": {
    "allow": [],
    "deny": []
  }
}
```

## 最佳实践

- **在项目根目录启动** - 让 Claude Code 能够读取项目文件，提供更准确的建议
- **使用 /help 命令** - 查看所有可用的斜杠命令和功能
- **使用 /clear 命令** - 清空上下文，开始新的对话（减少 token 消耗）
- **善用 @file 引用** - 在对话中使用 @filename 快速引用项目文件
- **备份重要文件** - Claude Code 可以直接修改文件，操作前建议使用 Git 管理版本
