## Step 1: Install Cursor

1. Visit [https://cursor.com](https://cursor.com) to download the installer for your system
2. Install and open Cursor

## Step 2: Configure API

**Via Settings Interface**

Configure the following:

- **OpenAI API Key**: Enter your API key
- **Enable Override OpenAI Base URL**: `https://api.example.com`

After configuration, it's recommended to disable Cursor's built-in cloud models (uncheck them) in the Models page to avoid extra charges.

## Step 3: Start Using

- Select the models you want to use from the model list
- **Inline Generation/Edit**: Press `Ctrl+K` (Mac: `Cmd+K`) to generate or edit code inline
- **AI Chat Panel**: Press `Ctrl+L` (Mac: `Cmd+L`) to open the AI chat panel
- **Composer Mode**: Press `Ctrl+I` (Mac: `Cmd+I`) to open Composer for multi-file edits
