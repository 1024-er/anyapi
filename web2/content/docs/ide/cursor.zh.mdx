## Step 1: 安装 Cursor

1. 访问 [https://cursor.com](https://cursor.com) 下载对应系统的安装包
2. 安装并打开 Cursor

## Step 2: 配置 API

**通过设置界面配置**

填写以下配置：

- **OpenAI API Key**: 输入你的 API 密钥
- **打开 Override OpenAI Base URL**: `https://api.example.com`

配置完成后，建议在 Models 页面关闭 Cursor 自带的云端模型（取消勾选），仅使用自定义 API 以避免额外费用。

## Step 3: 开始使用

- 在上方的模型列表中，勾选你需要使用的模型
- **内联生成/编辑**: 按 `Ctrl+K`（Mac: `Cmd+K`）在编辑器内直接生成或编辑代码
- **AI 聊天面板**: 按 `Ctrl+L`（Mac: `Cmd+L`）打开右侧 AI 聊天面板
- **Composer 模式**: 按 `Ctrl+I`（Mac: `Cmd+I`）打开 Composer，可跨多个文件进行修改
