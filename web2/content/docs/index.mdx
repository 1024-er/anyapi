## What is Any API?

Any API is an AI API proxy platform built for developers. It provides stable network routing and convenient payment methods, making it easy to use the world's top AI programming assistants.

### Supported AI Tools

- [Claude Code](/docs/installation/macos) — Anthropic's official AI coding assistant with terminal interaction, code generation, and project understanding (Recommended)
- [Codex CLI](/docs/installation/macos) — OpenAI's command-line AI coding tool with code completion and generation
- [Gemini CLI](/docs/installation/macos) — Google's AI coding assistant with multimodal understanding and code generation

### Getting Started in 4 Steps

1. **Get an API Key** — Register and log in, get your API key from the console → [Get Key](/console/token)
2. **Install AI Tools** — Install the AI programming tool for your operating system
3. **Configure Proxy** — Set environment variables to route API requests through the proxy
4. **Start Using** — Launch the tool and enjoy smooth AI-assisted programming
